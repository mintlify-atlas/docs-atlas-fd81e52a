# xpander.ai SDK

The xpander.ai SDK is the official Python library for the xpander.ai **Backend-as-a-Service (BaaS)** platform. It gives you everything you need to create, deploy, and orchestrate AI agents — with production-grade tools, knowledge bases, event handling, and real-time streaming built in.

## Async/sync support

Every public method in the SDK is available in both async and sync flavors:

```python
# Async (recommended for production)
agent = await Agent.aload("agent-id")
task = await agent.acreate_task(prompt="Analyze this data")

# Sync (convenient for scripts and notebooks)
agent = Agent.load("agent-id")
task = agent.create_task(prompt="Analyze this data")
```

## Framework support

The SDK integrates with major AI agent frameworks:

- **Agno** — via `pip install xpander-sdk[agno]`
- **Strands Agents** — included in the base install
- **OpenAI Agents** — included in the base install

## Requirements

- Python 3.9+
- An xpander.ai account with an API key
- An organization ID

<Note>
  All API operations require a valid `XPANDER_API_KEY` and `XPANDER_ORGANIZATION_ID`. See the [Authentication guide](/authentication) for setup instructions.
</Note>
