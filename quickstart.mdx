<Steps>
  <Step title="Install the SDK">
    Install `xpander-sdk` from PyPI:

    <CodeGroup>

    ```bash pip
    pip install xpander-sdk
    ```

    ```bash agno framework
    pip install xpander-sdk[agno]
    ```

    ```bash requirements.txt
    pip install -r requirements.txt
    ```

    </CodeGroup>

    See [Installation](/installation) for optional extras and full dependency details.
  </Step>

  <Step title="Set your credentials">
    Export your API key and organization ID as environment variables:

    ```bash
    export XPANDER_API_KEY="your-api-key"
    export XPANDER_ORGANIZATION_ID="your-org-id"
    ```

    <Note>
      Find your API key and organization ID in the [xpander.ai dashboard](https://app.xpander.ai). See [Authentication](/authentication) for all configuration methods.
    </Note>
  </Step>

  <Step title="Configure the SDK">
    Create a `Configuration` object. When environment variables are set, you can call it with no arguments:

    ```python
    from xpander_sdk import Configuration

    # Reads XPANDER_API_KEY and XPANDER_ORGANIZATION_ID automatically
    config = Configuration()

    # Or pass credentials explicitly
    config = Configuration(
        api_key="your-api-key",
        organization_id="your-org-id",
        base_url="https://inbound.xpander.ai"
    )
    ```
  </Step>

  <Step title="Load an agent and create a task">
    Use the `Agents` module to load an existing agent, then create a task against it:

    ```python
    import asyncio
    from xpander_sdk import Configuration, Agents

    async def main():
        config = Configuration()
        agents = Agents(configuration=config)

        # List all available agents
        agent_list = await agents.alist()
        print(f"Found {len(agent_list)} agents")

        # Load a specific agent by ID
        agent = await agents.aget("agent-id")
        print(f"Loaded agent: {agent.name}")

        # Create and execute a task
        task = await agent.acreate_task(
            prompt="Help me analyze this data",
            file_urls=["https://example.com/data.csv"]
        )
        print(f"Task created: {task.id}")
        print(f"Status: {task.status}")

    asyncio.run(main())
    ```
  </Step>

  <Step title="Stream task events">
    Enable real-time streaming to receive events as the agent processes the task:

    ```python
    import asyncio
    from xpander_sdk import Configuration, Agents

    async def main():
        config = Configuration()
        agents = Agents(configuration=config)
        agent = await agents.aget("agent-id")

        # Create a task with streaming enabled
        task = await agent.acreate_task(
            prompt="complex analysis task",
            events_streaming=True
        )

        # Stream events as the agent runs
        async for event in task.aevents():
            print(f"Event type: {event.type}")
            print(f"Event data: {event.data}")

    asyncio.run(main())
    ```

    <Tip>
      Streaming uses Server-Sent Events (SSE). Events include tool calls, status updates, and completion signals.
    </Tip>
  </Step>
</Steps>
